# React Todo List:

### This is a super simple Todo App built using :
1. React.js.
2. React Components.
3. React Hooks.
4. Local Storage.


## 😃 Features:
- ➕ Add todos.
- 🗑️ Delete todos.
- ⏸ Save todos to Local Storage when the state changes.
- ▶️ Load the todos from Local Storage when the site is loaded again.
